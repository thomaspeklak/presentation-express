Express.js

web application framework for node

Express ain't Node.js

Node's HTTP module

  • Very low level.
  • No cookie handling or parsing.
  • No session support built-in.
  • No routing built-in.
  • No static file serving.

Connect

a middleware framework for node

  • csrf
  • compress
  • basicAuth
  • bodyParser
  • json
  • urlencoded
  • multipart
  • timeout
  • cookieParser
  • session
  • cookieSession
  • methodOverride
  • responseTime
  • staticCache
  • static
  • directory
  • vhost
  • favicon
  • limit
  • query
